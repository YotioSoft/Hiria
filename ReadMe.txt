1. はじめに
　この度は、「非リアが手を握るのはだめですか」をダウンロードしていただきありがとうございます。
このゲームは非リア充がリア充を蹴飛ばすゲームですので、プレイの際はリア充の方はお気をつけください。

2. ルール
　非リア充を十字キー操作して、リア充を蹴飛ばします。リア充は画面左側から流れてきます。
仲間非リア充に接触すると、非リア充と手をつなぐことができます。
非リア充と手を繋ぐと、リア充に当たる面積が広がり、リア充を蹴飛ばしやすくなります。
岩につまづくと、つまづいた非リア充が飛ばされ、仲間が一人減ります。
赤い三角印の下にいる非リア充がプレイヤーです。プレイヤーが岩につまづくと、ゲームが終了します。
　制限時間はありません。

3. 点数
・リア充を蹴飛ばす +5点
・非リア充と仲間になる +10点
